Descrição da Classe e Funcionalidades

A classe Sistema é responsável por gerenciar as operações relacionadas aos alunos, incluindo cadastro, edição de nomes e listagem de informações.

Funcionalidades Principais:

  • Cadastro de alunos
  • Edição de nomes de alunos, incluindo tratamento de espaços
  • Listagem de informações de alunos

Escopo dos Testes

Funcionalidades a serem testadas:

  • Edição de nomes com espaços
  • Cadastro de novos alunos
  • Listagem de alunos cadastrados

Funcionalidades fora do escopo:

  • Remoção de alunos
  • Alteração de dados além do nome (ex: endereço, telefone)

Critérios de Sucesso e Falha

Critérios de Sucesso:

  • Resultados corretos: Todas as operações devem retornar os resultados corretos para uma variedade de entradas.
  • Validação: A classe deve lançar validações apropriadas em casos de erro, como entradas inválidas.

Critérios de Falha:

  • Resultados Incorretos: Se a classe retornar resultados divergentes em qualquer operação, isso será considerado uma falha.
  • Validação: A falta de validações apropriadas será considerada uma falha.
